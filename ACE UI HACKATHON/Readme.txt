‘Step into Fairy kingdom’ is a Birthday party theme provider which provides various themes for the birthday parties especially girls.
The page is developed using html and pure css where the sass pre-compiler is used.It requires the internet connection.
It can work best with Google chrome.



The features of the page are following:
1. Animated navigation icons where all the icons are customized one which uses of minimal elements and not of any font awesome icons.

2. The themes for the birthday parties are displayed using flex boxes which has animated effects.3. The Favorite icon and theme name are displayed at the bottom of each flex box.

4. The page is responsive.
